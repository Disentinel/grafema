# User Request

## Task
Разобраться почему NodeCreationValidator не отлавливает создание нод из разных мест.

## Context
- Linear issue: REG-94 (CLI interface development)
- При тестировании explore - UI не показывает важные вещи
- Причина: ноды создаются в разных местах, не всегда с правильными атрибутами
- Добавлена гарантия (NodeCreationValidator) но она ничего не ловит

## Requirements
1. Выяснить почему гарантия не отлавливает создание нод из разных мест
2. Заставить её работать
3. Используя гарантию как линтер, завести отдельные задачи на рефакторинг каждого типа нод в Linear
4. Отчитаться в чём была проблема и какой подход к разработке её вызвал
5. Предложить изменение workflow чтобы такого не повторялось
