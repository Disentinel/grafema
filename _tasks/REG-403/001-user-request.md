# REG-403: User Request

## Request

Добавить в config поле `version` и убедиться что оно совпадает с версией графемы (иначе кидать алярму). Защита от breaking changes схемы конфига.

## Key Requirements

1. Config должен содержать `version` field
2. При загрузке конфига — проверять совместимость с текущей версией Grafema
3. При несовпадении — понятное сообщение об ошибке
4. Обратная совместимость — конфиги без `version` должны работать
