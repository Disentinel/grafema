# REG-338: Rename RFDB to Rega Flow Database

## Задача

Переименовать RFDB (Reginaflow Database) в **Rega Flow Database**.

## Обоснование названия

**Rega Flow** — название с намеренной многослойностью:

* **Rega** (רגע) — «мгновение» на иврите; также краткая форма имени Regina
* **Flow** — поток выполнения, данных и причинно-следственных переходов

### Инженерный смысл

**Мгновенный поток** — система, оптимизированная для:

* Сверхбыстрой навигации
* Hot-path сценариев
* Embedded DB
* Агентных рантаймов
* IDE-инструментов

### Семантический смысл

**Поток мгновений** — модель, где система представлена не статическим хранилищем, а цепочкой дискретных состояний, связанных в граф причин, инвариантов и переходов.

* **RFDB** фиксирует мгновения как узлы и связи
* Позволяет восстанавливать смысл, контекст и исполнение без реконструкции из текста
* **RFDB** отвечает на «что истинно», **Rega Flow** — «как это происходит»
